Available Data

WOF data can be chunked into three different categories: administrative, venue, and other. Explore those categories below:

  • Administrative Data - all administrative placetypes (all the places between and inclusive of continents to microhoods) for the entire world
  • Venue Data - all venues for the entire world
  • Other Data - all postalcodes, constituencies, intersections, pics, and brands for the entire world

Administrative Data

Administrative data is located in the whosonfirst-data repository. This repository contains all administrative placetypes (all the places between and inclusive of continents to microhoods) for the entire world.

You can find data for the following placetypes in the whosonfirst-data repository:

  • continent
  • empire
  • country
  • macroregion
  • region
  • macrocounty
  • county
  • metro area
  • locality
  • macrohood
  • neighbourhood
  • microhood

Find the whosonfirst-data repo here.

Venue Data

Venue data is located in a number of different repos. The whosonfirst-data-venue repository can serve as a starting point for venue data in Who's On First. Keep in mind there is no actual in this repository. The actual data has been stored in per-country (and in some case per country-region) repositories.

Naming conventions for these repositories follow:

	whosonfirst-data-venue + "-" + WHOSONFIRST_COUNTRY + "-" + WHOSONFIRST_SUBDIVISION

Two of these repos are:

  • whosonfirst-data-venue-us-ca: venues in California, USA
  • whosonfirst-data-venue-ca: venues in Canada

Find the whosonfirst-data-venue repo here.
