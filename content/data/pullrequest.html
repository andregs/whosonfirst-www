Pull Request Template

Who's On First (WOF) relies on your local knowledge, interest, and expertise to contribute to an ever-improving database of places.

When submitting a Pull Request (PR) to the whosonfirst-data repository, we kindly ask you to use this template in your PR description and replace the website links and Who's on First IDs appropriately:

  • What problem does this address?

  • What did you see in Who's on First?

  • What did you expect to see?

  • Overview of edits and methodology:

  • How was the data updated to address the problem?

  • How did you perform the edit? (Someone else might want repeat your fix elsewhere, play your fix forward.)

  • Direct link to your source or references (.shp or .geojson file) that you used for new administrative geometries:

  • https://www.directlink.gov/file.geojson

  • Direct link to the open license file that is associated to that source file:

  • https://www.directlink.gov/license.txt

  • List of all WOF records that were deprecated:

  • 012345678, 123456789, 234567890, etc.

  • List of all WOF records that were given supersedes or supersedes_by values:

  • 012345678, 123456789, 234567890, etc.

  • List of all WOF records that changed placetype :

  • 012345678, 123456789, 234567890, etc.

  • List of all WOF records that were updated in any way, shape, or form:

    • 012345678, 123456789, 234567890, etc.
