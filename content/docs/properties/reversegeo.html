reversegeo

latitude

Typically derived from mapshaper, reversegeo:latitude represents the latitude value to use when reverse geocoding a record.

longitude

Typically derived from mapshaper, reversegeo:longitude represents the longitude value to use when reverse geocoding a record.

polygon

This optional string property indicates which of the feature's named geometries to use for point-in-polygon reverse geocoding tasks. The property value namees a polygon source prefix, which is typically stored in an alt-geometry file.

The reversegeo:polygon value is represented as {source} in the following filename expansion:

85922583-alt-{source}-extras.geojson
